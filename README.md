# AlphaSignal - Premium Financial Intelligence Service

## Overview
AI-powered market intelligence combining Twitter sentiment, contradiction detection, and real-time narrative analysis.

## Revenue Tiers
- **Free**: Weekly digest, basic Twitter threads
- **Pro ($49/mo)**: Daily briefings, contradiction alerts, watchlist monitoring
- **Elite ($199/mo)**: Real-time alerts, custom research, 1:1 consultations

## Tech Stack
- TwitterAPI.io for social sentiment
- Exa AI for deep research
- Cron jobs for automated delivery
- Telegram/WhatsApp for distribution
- Stripe for payments
